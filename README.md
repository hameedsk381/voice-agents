# Voice Agents Project

This is a monorepo containing both frontend and backend components for the voice agents platform.

## Project Structure

- `frontend/` - Next.js frontend application
- `backend/` - FastAPI backend application  
- `docker-compose.yml` - Docker configuration for the entire project

## Setup Instructions

### Prerequisites
- Docker and Docker Compose
- Node.js (for frontend development)
- Python 3.8+ (for backend development)

### Running the Project

Using Docker Compose:
```bash
docker-compose up
```

## Development

Each subdirectory (frontend, backend) has its own git repository for more granular version control of individual components.
